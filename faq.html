El dispositivo usa sensores para medir metales pesados, la temperatura del agua, y la cantidad de agua que fluye a través de él. Estos sensores recogen datos sobre la calidad del agua, que luego se utilizan para determinar si el agua es segura para consumir.


Para comunicarse y enviar los datos que recoge, el dispositivo probablemente utilice protocolos estándar de comunicación de Internet como HTTP (Protocolo de Transferencia de Hipertexto) o MQTT (Message Queuing Telemetry Transport). MQTT es muy común en dispositivos del Internet de las Cosas (IoT) por su eficiencia en el envío de datos entre dispositivos y sistemas en tiempo real.


Sí, el dispositivo necesita conexión a Internet para enviar los datos recogidos a un servidor donde se almacenan y procesan. Estos datos luego son accesibles a través del portal web donde los usuarios pueden ver la información sobre la calidad del agua.


Para conectar el dispositivo a Internet, sigue estos pasos:

1. Conecata tu dispositivo movil a la red indicada en la pantalla LCD.

2. Se habrirá una ventana en el navegador
Selecciona "Connect Wifi" y seleciona la red a la que desees conectar el dispositivo.

3. Abre el portal web y activa las lecturas en "Enviar lecturas".
